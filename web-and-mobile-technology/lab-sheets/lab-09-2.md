At the initially we assign the cart count as zero. There for inside the useState bracket we input 0 . And the initial values for the stock we need to add a count like as follows.

```javascript

const [cartCount, setCartCount] = useState(0); // useState for cart

const [stock, setStock] = useState({          // useState for pet stock

  "Golden Retriever": 5,

  "Persian Cat": 3,

  "Parrot": 10

});

```

2. Then we need to create the function call addToCart (    ). Inside that function need to be included followings.

```javascript

function addToCart(petName) {             // Event handler

  if (stock[petName] > 0) {

    setCartCount(cartCount + 1);

    setStock({ ...stock, [petName]: stock[petName] - 1 });

  } else {

    alert(`${petName} is out of stock!`);

  }

}

```

## Task 6: Handling Events

We use the onClick event handler inside the button so that when the user clicks the button, React executes the given function (like adding a pet to the cart), allowing the app to respond to user actions.

```html

<button onClick={props.onAddToCart} disabled={props.stock === 0}>

  {props.stock > 0 ? "Add to Cart" : "Out of Stock"}

</button>

```

Explanation: This button calls the add to cart function when clicked, and it automatically disables itself and shows "Out of Stock" when the stock becomes zero, otherwise it shows "Add to Cart".
