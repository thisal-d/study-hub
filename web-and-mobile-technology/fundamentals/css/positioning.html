CSS Position Examples

1️⃣ Static (default)

static

➡️ Static is the default position — the element stays in the normal page flow.

2️⃣ Relative

relative

➡️ Relative moves slightly *from its normal place* but still takes up space in layout.

3️⃣ Absolute

absolute

➡️ Absolute is positioned *relative to the nearest positioned parent* (`.container` here).

4️⃣ Fixed

fixed

➡️ Fixed stays in one spot on screen (even if you scroll).
