CSS Float Layout Demonstration

Fixed Layout: Uses pixel widths. The layout does not change when resizing the window.

Fluid Layout: Uses percentage widths. The layout adjusts based on browser size.

1️⃣ Fixed Layout (Pixel-based)

Sidebar (200px wide)
Main Content (480px wide)

Try resizing your window — the layout width stays the same.

2️⃣ Fluid Layout (Percentage-based)

Sidebar (30%)
Main Content (70%)

Try resizing your browser — this layout shrinks and expands fluidly.
