# 🎓 SE2020 — Lab Test Practice System

> **Based on:** All lectures, lab sheets, and the official lab test notice  
> **Test format:** 2-hour lab test — complete & extend a MERN Item Manager project  

---

## 📋 What the Real Lab Test Expects

From the official notice, you will:
1. Receive a **starter MERN project** (separate frontend + backend)
2. **Complete** the given project
3. **Add a new field** (or fields) to extend it
4. **Test locally** using your own MongoDB Atlas connection string
5. **Push source code** to a public GitHub repository
6. **Deploy frontend** (Netlify / Vercel)
7. **Deploy backend** (Railway / Render)
8. **Connect** the deployed frontend to the deployed backend (update API URLs)

## ⚡ Quick Checklist Before the Test

- [ ] GitHub account ready (can create public repos)
- [ ] MongoDB Atlas account ready (connection string obtainable)
- [ ] Node.js + npm installed and working
- [ ] VS Code ready
- [ ] Netlify or Vercel account (frontend deployment)
- [ ] Railway or Render account (backend deployment)
- [ ] Know how to `npm install`, `npm run dev`, `npm start`
- [ ] Know how to add `.env` files and update API URLs after deployment
