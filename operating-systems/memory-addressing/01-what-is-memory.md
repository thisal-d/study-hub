## The Street Analogy

Think of memory (RAM) as a **long street of numbered houses**.

```
┌──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┐
│  00  │  01  │  02  │  03  │  04  │  05  │  06  │  07  │
│      │      │      │      │      │      │      │      │
└──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┘
  Addr   Addr   Addr   Addr   Addr   Addr   Addr   Addr
   0      1      2      3      4      5      6      7
```

- Each **house** → one **memory location**
- Each **house number** → one **memory address**
- The **data stored inside** → the **contents** of that location

---

## Key Terms

| Term | Meaning |
|------|---------|
| **Memory Location** | One single slot where data is stored |
| **Memory Address** | The unique number that identifies a location |
| **Total Locations** | How many slots exist in the whole memory |

> **Key Point:** Every location gets its own unique address, starting from **0**.
