## What is a Word?

A **word** is how much data ONE memory location can hold.

```
  Word Size = 1 byte (default assumption)
  ┌────────┐
  │ 1 byte │  ← one memory location holds 1 byte
  └────────┘

  Word Size = 4 bytes
  ┌────────┬────────┬────────┬────────┐
  │ byte 0 │ byte 1 │ byte 2 │ byte 3 │  ← one location holds 4 bytes
  └────────┴────────┴────────┴────────┘
```

> **Rule of Thumb:** Unless told otherwise in a question, always assume **word size = 1 byte**.
