## The Master Equation

```
╔══════════════════════════════════════════════════════╗
║                                                      ║
║   Number of Locations  =  2 ^ (address width)        ║
║                                                      ║
║   Total Memory Size    =  Locations × Word Size      ║
║                                                      ║
╚══════════════════════════════════════════════════════╝
```

## Rearranged Forms (For Reverse Questions)

If you are given the total size and need to find the address bits:

```
  Number of Locations  =  Total Size ÷ Word Size

  2 ^ (Address Width)  =  Number of Locations
  (Find the address width by asking: 2 to what power equals the number of locations?)
```
