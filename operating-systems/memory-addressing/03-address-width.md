## Binary Addresses

Addresses are stored in **binary** (0s and 1s).

**Address width** = how many bits are needed to write the largest address.

```
  4-bit address can represent:
  0000 → address 0
  0001 → address 1
  0010 → address 2
  ...
  1111 → address 15

  Total unique addresses = 2^4 = 16
```

## The Rule

```
  n-bit address  →  2^n unique addresses (locations)
```

For example, a 10-bit address gives you $2^{10} = 1024$ unique locations.
