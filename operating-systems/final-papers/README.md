## 🖼️ Assets & Diagrams

Each paper includes 3 custom-generated diagrams stored in the [`assets/`](./assets/) folder:
1. **Q1 Scenario:** Visual representation of the synchronization problem (e.g., Attendance, Cinema, Blood Bank).
2. **Q2 Paging:** Memory layout showing logical pages, page tables, and physical frames.
3. **Q3 RAG:** Resource Allocation Graphs for deadlock analysis.
