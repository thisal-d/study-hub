## Data Types and Variables

In C, you must declare a variable and its type before using it. This tells the computer how much memory to reserve.

### How to initialize variables

> **How to initialize an `int` (Integer):**
> Used for whole numbers (e.g., 10, -5, 0).
> ```c
> int age = 20;
> int marks;
> ```

> **How to initialize a `float` or `double` (Decimal):**
> Used for numbers with decimal points (e.g., 95.5, 3.14). `double` is more precise than `float`.
> ```c
> float temperature = 36.5;
> double precise_score = 99.987;
> ```

### The `char` data type
Used to store a single character. Note that characters are surrounded by **single quotes**.

> **How to initialize a `char`:**
> ```c
> char grade = 'A';
> char gender = 'M';
> ```

## Input and Output (I/O)

### Output with `printf`
We use `printf` to display information. We use **format specifiers** (placeholders) to insert variables into the text.

**Common Format Specifiers:**
*   `%d` or `%i`: Integer
*   `%f`: Float
*   `%lf`: Double (for `scanf`, use `%lf`; for `printf`, `%f` is usually fine, but `%lf` is safe)
*   `%c`: Character
*   `%s`: String (text)

**Example:**
```c
int count = 10;
printf("The count is %d\n", count);
// Output: The count is 10
```

### Input with `scanf` and the `&` operator
To get input from the keyboard, we use `scanf`. This is where we first encounter the concept of **memory addresses**.

> **How to get an integer input:**
> ```c
> int number;
> scanf("%d", &number);
> ```

**âš ď¸Ź Important: Why the `&`?**
*   `number` is the **value** (e.g., 10).
*   `&number` is the **address** in memory where that value is stored.
*   `scanf` needs the **address** to know where to store the input value.
*   *Note: This concept is the precursor to Pointers.*

## Arithmetic and The Integer Division Trap

In **[Lab Sheet 1: Activity 3](../labsheets/1.md#activity-3)**, there is a critical observation regarding division.

**The Rule:**
If you divide an integer by an integer, C **truncates** (removes) the decimal part.

*   `50 / 100` results in `0` (Because 0.5 becomes 0).
*   `50 / 100.0` results in `0.5` (Because one operand is a decimal).

**Example Code:**
```c
int a = 50;
int b = 100;

// WRONG way for calculating percentage
float result1 = a / b;           // Result is 0.000000

// CORRECT way
float result2 = a / (b * 1.0);   // Result is 0.500000
float result3 = a / 100.0;       // Result is 0.500000
```

> [!TIP]
> **Summary & Quick Recap:**
> - **Variables:** Declare type before use (`int`, `float`, `char`).
> - **I/O:** Use `%d`, `%f`, `%s` with `printf` and `scanf`. Don't forget `&` for `scanf` (except strings!).
> - **The Math Trap:** Integer divided by integer is an integer. To get a decimal, make at least one number a float (`100.0`).
