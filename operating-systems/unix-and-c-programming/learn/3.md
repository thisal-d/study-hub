## Introduction
So far, our variables hold only one value. What if we need to store 100 marks? Defining 100 variables is impossible. **Arrays** solve this. Also, writing all code inside `main()` gets messy. **Functions** allow us to split code into reusable blocks.

---

## Arrays in C

An array is a collection of variables of the **same type** stored in contiguous memory locations.

### How to declare and initialize arrays

> **How to declare an array (e.g., size 5):**
> ```c
> int marks[5]; // Declares an array that can hold 5 integers
> ```

> **How to initialize with values:**
> ```c
> int scores[3] = {90, 85, 95};
> ```

### Accessing array elements
Array indices start at **0**. The last element is at index `size - 1`.

> **Accessing the first element:**
> ```c
> printf("%d", scores[0]); // Output: 90
> ```

> **Assigning a value to the second element:**
> ```c
> scores[1] = 80; // Changes 85 to 80
> ```

### Arrays and Loops
You almost always use a loop to handle arrays (See **[Lab Sheet 3: Activity 4](../labsheets/3.md#activity-4)**).

> **Inputting values into an array:**
> ```c
> int data[5];
> for (int i = 0; i < 5; i++) {
>     scanf("%d", &data[i]); // Don't forget the &
> }
> ```

### Algorithms: Finding Min/Max ([Lab 3: Activity 4](../labsheets/3.md#activity-4))
Finding the minimum or maximum value in an array is a very common exam requirement. You must write a loop that compares each element to the current min/max.

> **Logic for finding Maximum:**
> 1. Initialize `max` with the first element of the array.
> 2. Loop through the rest of the array.
> 3. If an element is greater than `max`, update `max`.
>
> ```c
> int data[5] = {15, 42, 8, 99, 23};
> int max = data[0]; // Start with the first element
> 
> for (int i = 1; i < 5; i++) { // Start loop from index 1
>     if (data[i] > max) {
>         max = data[i]; // Update max if a larger value is found
>     }
> }
> printf("Max is: %d\n", max);
> ```
> *For finding the minimum, just change the `>` to `<`.*

### Strings (Character Arrays)
In C, a string is simply an array of characters, ending with a special null character `'\0'`. This is heavily used in **[Lab Sheet 6](../labsheets/6.md) (Pipes)**.
*   **Important:** You do **NOT** use the `&` operator when scanning strings with `scanf`!

> **How to create and read a String:**
> ```c
> char name[50]; // Can hold up to 49 characters + 1 null character
> scanf("%s", name); // Notice NO '&' before name!
> ```

**`sizeof()` vs `strlen()`:**
*   `sizeof(name)` gives the total *capacity* of the array (e.g., 50 bytes).
*   `strlen(name)` gives the actual length of the text inside the array.
*   **Critical Exam Trap:** To use `strlen()`, you MUST `#include <string.h>`. Without this, compiling from memory in an exam will fail.

## Functions in C

A function is a block of code that performs a specific task. It helps avoid code repetition (See **[Lab Sheet 3: Activity 5](../labsheets/3.md#activity-5)**).

### How to define a function

> **Syntax:**
> ```c
> return_type function_name(parameters) {
>     // code
>     return value; // (if return_type is not void)
> }
> ```

### Return types (`void` vs `int/float`)

1.  **`void`**: The function does not return a value. It just *does* something (like print).
    > ```c
    > void printHello() {
    >     printf("Hello!\n");
    >     // No return statement needed
    > }
    > ```

2.  **`int` or `float`**: The function calculates and sends a result back.
    > ```c
    > int add(int a, int b) {
    >     int sum = a + b;
    >     return sum; // Sends sum back to main
    > }
    > ```

## Passing Arrays to Functions

This is a critical concept for your lab test. When passing an array to a function, you usually pass the **array name** and the **size**.

**Note:** You don't need `&` when passing an array (we will explain why in the Pointers README).

> **Example: Function to find the sum of an array.**
> ```c
> // Function Definition
> int calculateSum(int arr[], int size) {
>     int total = 0;
>     for (int i = 0; i < size; i++) {
>         total = total + arr[i];
>     }
>     return total;
> }
>
> int main() {
>     int numbers[3] = {10, 20, 30};
>     int result = calculateSum(numbers, 3); // Call the function
>     printf("Total: %d", result);
>     return 0;
> }
> ```

> [!TIP]
> **Summary & Quick Recap:**
> - **Arrays:** Stores multiple same-type values. Starts at index `0`. Use loops to access items.
> - **Strings:** Char arrays ending with `\0`. No `&` in `scanf`. Include `<string.h>` for `strlen()`.
> - **Functions:** Reusable code. `void` returns nothing. Others return a value (`int`, `float`).
> - **Passing Arrays:** Pass array name without `&`. Example: `myFunction(arr, size);`

## Practice Questions

### 🌱 Small Coding Questions
1.  **Array Max:** Write a program that declares an array of 3 integers. Manually (without a loop) assign values. Write an `if` statement to check which one is the largest and print it.
2.  **Function Call:** Write a function named `square(int n)` that takes an integer and prints its square (n * n). Call this function from `main`.

### 🚀 Larger Coding Question
**Problem:** Write a C program that calculates the average of 5 numbers entered by the user. You **must** use functions.

1.  Create an array `float values[5]`.
2.  Write a function `void getInputs(float arr[])` that uses a loop to ask the user for inputs and store them in the array.
3.  Write a function `float findAverage(float arr[])` that calculates and returns the average.
4.  Print the average in `main`.

**Sample Output:**
```text
Enter value 1: 10
Enter value 2: 20
Enter value 3: 30
Enter value 4: 40
Enter value 5: 50
The average is: 30.000000
```
