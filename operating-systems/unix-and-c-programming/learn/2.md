## Introduction
Programs aren't just linear lists of instructions. They need to make decisions (e.g., "Pass or Fail") and repeat tasks (e.g., "Input 5 numbers"). This guide covers the syntax found in **[Lab Sheet 2](../labsheets/2.md)** and **[Lab Sheet 3](../labsheets/3.md)**.

---

## Selection Statements (Decision Making)

### The `if-else` statement
This allows your program to execute different code blocks based on conditions.

> **Basic Syntax:**
> ```c
> if (condition) {
>     // Code runs if condition is true
> } else {
>     // Code runs if condition is false
> }
> ```

> **Example (Pass/Fail from [Lab 2](../labsheets/2.md)):**
> ```c
> if (average > 45) {
>     printf("Pass");
> } else {
>     printf("Fail");
> }
> ```

### Logical Operators
In **[Lab Sheet 2: Activity 3](../labsheets/2.md#activity-3)**, you need to check two things at once (Gender AND Age).

*   `&&` (AND): Both must be true.
*   `||` (OR): At least one must be true.

> **How to check multiple conditions:**
> ```c
> if (age >= 65 && gender == 'M') {
>     printf("Senior Male citizen");
> }
> ```

### The `switch-case` statement
Used when comparing one variable against many specific constant values (integers or chars). **Don't forget the `break`!**

> **Syntax ([Lab 2: Activity 5](../labsheets/2.md#activity-5)):**
> ```c
> int choice = 1;
> switch (choice) {
>     case 1:
>         printf("Hello");
>         break; // Stops the switch here
>     case 2:
>         printf("Welcome");
>         break;
>     default:
>         printf("Invalid choice");
> }
> ```

## Iteration Statements (Loops)

Loops allow you to repeat code without writing it multiple times.

### The `while` loop
Checks the condition *before* running the code. Use it when you don't know exactly how many times the loop will run.

> **Syntax ([Lab 3: Activity 1](../labsheets/3.md#activity-1)):**
> ```c
> int i = 1;
> while (i <= 5) {
>     printf("%d\n", i);
>     i++; // Don't forget to increment!
> }
> ```

### The `for` loop
Best when you know exactly how many times to loop (e.g., "Loop 5 times" or "Loop through an array of size 10").

> **How to write a `for` loop:**
> ```c
> // (initialization; condition; update)
> for (int i = 1; i <= 5; i++) {
>     printf("%d\n", i);
> }
> ```

**Pattern Printing Tip ([Lab 3: Activity 3](../labsheets/3.md#activity-3)):**
To print patterns like:
```text
1
1 2
1 2 3
```
You need **nested loops**: an outer loop for the rows, and an inner loop for the numbers in each row.

## Common Pitfalls

1.  **`=` vs `==`:**
    *   `if (x = 5)` is **WRONG**. This assigns 5 to x. It is always "true".
    *   `if (x == 5)` is **CORRECT**. This checks if x is 5.

2.  **The Semicolon Trap:**
    *   `if (x > 5);` { ... } -> The semicolon ends the statement immediately. The code block inside `{}` will *always* run.

> [!TIP]
> **Summary & Quick Recap:**
> - **Selection:** `if-else` for choices, `switch-case` for specific values (remember the `break`!).
> - **Logical:** `&&` (AND), `||` (OR).
> - **Loops:** `while` (don't know how many times), `for` (know exact times).
> - **Pitfalls:** `==` is for checking, `=` is for assigning. No semicolons after `if` or `for` loop conditions.

## Practice Questions

### 🌱 Small Coding Questions
1.  **Grade Checker:** Write a program that asks for a mark. If the mark is 75 or above, print "Distinction". Otherwise, print "No Distinction".
2.  **Loop Conversion:** Convert the following `while` loop into a `for` loop.
    ```c
    int k = 0;
    while (k < 3) {
        printf("%d", k);
        k++;
    }
    ```

### 🚀 Larger Coding Question
**Problem:** Write a C program to print the multiplication table of a number entered by the user.
*   Ask the user to input a number `n`.
*   Use a loop to print the table from 1 to 10.

**Sample Output:**
```text
Enter a number: 5
5 x 1 = 5
5 x 2 = 10
...
5 x 10 = 50
```
