# Question 7: The Secret Pipe

## 📝 Problem Statement
The parent process creates a pipe and successfully forks a child. The **Parent process** writes a secret value (integer `100`) to the pipe. The **Child process** reads the value from the pipe and prints "Top Secret Code Received: 100".

## 💡 Expected Output
```text
[Child] Top Secret Code Received: 100
```
