# Question 8: Bi-directional Math Pipe

## 📝 Problem Statement
Create two pipes for bi-directional communication. The **Parent process** sends an integer to the **Child process**. The child squares that number and sends the result back to the Parent. Parent prints the final result.

## 💡 Expected Output
```text
Parent: Sending number 4...
Child: Squaring...
Parent: Squared result received = 16
```
