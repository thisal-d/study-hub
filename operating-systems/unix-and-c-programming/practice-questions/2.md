# Question 2: Pointer-based String Length

## 📝 Problem Statement
Write a C program to input a string from the user. Create a function `int getLength(char *ptr)` that uses a pointer to traverse the string and return its length **WITHOUT** using the standard `strlen()` function.

## 💡 Expected Output
```text
Enter a string: hello
Length is: 5
```
