# Question 9: Threaded Array Sum

## 📝 Problem Statement
Write a program that inputs 4 integers. Create a single thread and pass the entire array to it using a `void*` pointer. The thread should calculate the sum of the array and print "Array sum via thread: X".

## 💡 Expected Output
```text
Enter 4 integers: 10 20 30 40
Array sum via thread: 100
```
