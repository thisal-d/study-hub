# Question 1: Array Statistics

## 📝 Problem Statement
Write a C program to input 5 integers into an array. Create a function `void calculateStats(int arr[], int size)` that finds and prints the **Sum**, **Maximum**, and **Average** of the numbers.

## 💡 Expected Output
```text
Enter 5 numbers: 10 20 5 15 30
Sum: 80
Max: 30
Avg: 16.00
```
