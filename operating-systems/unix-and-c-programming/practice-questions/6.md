# Question 6: Multi-Threaded IDs

## 📝 Problem Statement
Write a program that creates 3 separate threads. Each thread should receive its "serial number" (1, 2, or 3) as a thread argument and print "Hello from Thread #X", where X is its serial number.

## 💡 Expected Output
```text
Hello from Thread #1
Hello from Thread #2
Hello from Thread #3
```
