# Question 4: The Safe Exec

## 📝 Problem Statement
Write a program that asks the user for a command path (e.g., `/bin/ls`). Fork a child. In the child, use `execl()` to run that command. In the parent, wait for the child and print "Command execution finished".

## 💡 Expected Output
```text
Enter command path: /bin/date
Fri Mar 6 00:00:00 UTC 2026
Command execution finished.
```
