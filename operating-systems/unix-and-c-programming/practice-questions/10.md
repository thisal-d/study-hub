# Question 10: Process-Thread Hybrid

## 📝 Problem Statement
The parent process forks a child. Inside the **Child process**, create a thread that prints "Greetings from the thread!". The main child process should join the thread, then exit. The parent should wait for the child and print "Operation Complete".

## 💡 Expected Output
```text
Greetings from the thread!
Operation Complete
```
