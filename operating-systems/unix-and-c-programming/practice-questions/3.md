# Question 3: Process Identification

## 📝 Problem Statement
Create a program that forks a child process. The **Child process** should print its own PID and its parent's PID. The **Parent process** should wait for the child to finish and then print "[Parent] Child process finished execution".

## 💡 Expected Output
```text
[Child] My PID: 1234, Parent PID: 1233
[Parent] Child process finished execution.
```
