## Arrays in C
An array is a collection of values of the same data type stored in continuous memory locations and accessed using a single name.

**Declaration Example:**
`int numbers [3] = {10, 20, 30};`

## Functions in C
A function is a block of code that performs a specific task. It divides a program into smaller parts, avoids repetition, and increases reusability.

**Example:**
```c
int add(int a, int b) {
    return a + b;
}

#include <stdio.h>

int main() {
    int result = add(10, 20);
    printf("%d\n", result);
    return 0;
}
```
