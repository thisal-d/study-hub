## Data types and variables in C
In C, variables directly represent memory locations. There are three primitive data types:
1.  **Integer:** `int`
2.  **Decimal:** `float`, `double`
3.  **Character:** `char`

Variables must be declared before use. Names are case sensitive and keywords are not allowed.

**Example Code:**
```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = 20;
    int sum;
    sum = a + b;
    printf("Sum = %d", sum);
    return 0;
}
```
*Note: `%d` is a format specifier (placeholder) for an integer.*
