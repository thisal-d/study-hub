**Objectives:** Understand UNIX POSIX Pthread management concepts using C language.

### Activity 1
Consider the below program and answer the questions.
```c
#include <stdio.h>
#include <stdlib.h>
#include <pthread.h>

void* print_message(void* arg) {
    printf("Hello from the thread!\n");
    pthread_exit(NULL);
}

int main() {
    pthread_t thread_id;
    if (pthread_create(&thread_id, NULL, print_message, NULL) != 0) {
        perror("Thread creation failed");
        return 1;
    }
    pthread_join(thread_id, NULL);
    printf("Thread has finished execution.\n");
    return 0;
}
```
a. What is the output you can see in the terminal?
b. What are the functions you found in the above program?
c. Briefly describe the above program.

*Note: Compile with `-lpthread` flag: `gcc -o program program.c -lpthread`*

### Activity 3
Consider the program given below and answer the questions.
```c
#include <stdio.h>
#include <pthread.h>

typedef struct {
    int a;
    int b;
} Numbers;

void* calculate(void* arg) {
    Numbers* nums = (Numbers*)arg;
    int sum = nums->a + nums->b;
    printf("Sum = %d\n", sum);
    return NULL;
}

int main() {
    pthread_t t;
    Numbers n;
    n.a = 10;
    n.b = 20;
    pthread_create(&t, NULL, calculate, &n);
    pthread_join(t, NULL);
    return 0;
}
```
a. What is the purpose of the `typedef struct` in this program?
b. Why must the thread function parameter be of type `void*`?
c. Explain the following line: `Numbers* nums = (Numbers*)arg;`
