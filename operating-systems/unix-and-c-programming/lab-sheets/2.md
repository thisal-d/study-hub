## Switch-case statements
A switch statement is a selection control structure used to execute one block of code among many, based on the value of an expression.

**Syntax:**
```c
switch(expression) {
    case constant1:
        statements;
        break;
    case constant2:
        statements;
        break;
    default:
        statements;
}
```

**Example:**
```c
#include <stdio.h>

int main() {
    int x = 2;
    switch (x) {
        case 1: printf("One");
            break;
        case 2: printf("Two");
            break;
        default: printf("Other");
    }
    return 0;
}
```
