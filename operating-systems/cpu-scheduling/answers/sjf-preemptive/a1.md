### **Solution to Practice Question 1: Preemptive SJF (SRTF)**

**How to Solve:**
1.  **Check at every arrival**: When a new process arrives, compare its burst time with the *remaining* burst time of the currently running process.
2.  **Preempt if shorter**: If the new arrival is shorter, stop the current process and start the new one.
3.  **Completion**: When a process finishes, pick the one with the shortest remaining time from the ready queue.

**Step-by-Step Execution:**
1.  **t=0**: $P_1$ (8) enters and runs.
2.  **t=1**: $P_2$ (4) arrives. $P_1$ remaining is 7. $P_2(4) < P_1(7)$. **Preempt $P_1$**, start $P_2$.
3.  **t=2**: $P_3$ (9) arrives. $P_2$ remaining is 3. $P_2(3) < P_3(9)$. Continue $P_2$.
4.  **t=3**: $P_4$ (5) arrives. $P_2$ remaining is 2. $P_2(2) < P_4(5)$. Continue $P_2$.
5.  **t=5**: $P_2$ finishes. Queue: $P_4(5), P_1(7), P_3(9)$. Smallest is $P_4$. Start $P_4$.
6.  **t=10**: $P_4$ finishes. Queue: $P_1(7), P_3(9)$. Smallest is $P_1$. Resume $P_1$.
7.  **t=17**: $P_1$ finishes. Queue: $P_3(9)$. Start $P_3$.
8.  **t=26**: $P_3$ finishes.

**Gantt Chart:**
`| P1 (0-1) | P2 (1-5) | P4 (5-10) | P1 (10-17) | P3 (17-26) |`
